# House Rent Prediction

## Overview
This project predicts house rental prices based on various property features such as BHK, size, furnishing status, and city. The goal is to build a regression model that can estimate rent accurately using structured housing data.

## Dataset
The dataset used for this project was obtained from Kaggle and contains information about rental properties including location, number of bedrooms, area size, and furnishing type.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Data Preprocessing
- Handling missing values
- Encoding categorical variables
- Feature selection
- Train-test split

## Model
A Linear Regression model was implemented to predict rental prices.

## Evaluation Metrics
The model was evaluated using:
- R² Score
- Mean Absolute Error (MAE)

## How to Run
1. Clone the repository.
2. Install the required dependencies using:
   pip install -r requirements.txt
3. Open and run the notebook inside the `notebook` folder.
